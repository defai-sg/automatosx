# CTO Tools and Frameworks

## Strategic Planning & Portfolio Governance
- **Workboard / Jira Align**: OKR tracking, initiative roadmaps, dependency visualisation across tribes
- **Productboard / Aha!**: Strategic prioritisation, capacity planning, and roadmap communication to executives
- **LeanIX / Ardoq**: Enterprise architecture inventory, capability mapping, and technology lifecycle oversight

## Engineering Execution & Quality
- **GitHub Enterprise / GitLab**: Source governance, policy enforcement, audit trails, and velocity analytics
- **CircleCI / Buildkite / Harness**: Enterprise CI/CD with progressive delivery, policy-as-code, and rollback automation
- **SonarQube / Snyk / Trivy**: Code quality, security scanning, and supply chain risk mitigation baked into pipelines
- **LaunchDarkly / Split**: Feature flag orchestration, experimentation, and safe rollout controls

## Observability & Operational Excellence
- **Datadog / New Relic / Grafana Cloud**: Unified metrics, tracing, and logging with SLO dashboards for every service
- **PagerDuty / Opsgenie**: Incident routing, on-call health monitoring, post-incident analytics
- **Jeli / Rootly**: Post-incident review automation and resilience programme reporting
- **CloudZero / Finout**: FinOps insight for unit economics, cost anomaly detection, and architectural ROI analysis

## Security & Compliance Posture
- **Drata / Vanta**: Continuous control monitoring, SOC 2 / ISO 27001 readiness evidence management
- **AWS Security Hub / Azure Defender**: Cloud posture visibility, threat intelligence integration, compliance checks
- **1Password / HashiCorp Vault**: Secrets governance, credential rotation, and Just-In-Time access policies
- **Tines / Torq**: Security automation and response workflows orchestrated with policy guardrails

## Leadership Communication & Knowledge Management
- **Notion / Confluence**: Architecture decision records, technology strategy narratives, operating models
- **Coda / Google Workspace**: Quarterly planning packs, hiring budget trackers, investor-ready reporting
- **Miro / FigJam**: Architecture review boards, capability mapping, future-state ideation with distributed teams
- **Slab / Guru**: Curated engineering handbook and leadership enablement resources accessible org-wide
