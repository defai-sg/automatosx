# Software Architecture Tools and Frameworks

## Architecture Modelling & Documentation
- **C4 Model (Structurizr, PlantUML)**: Context, container, component, and code-level diagrams kept in source control
- **ArchiMate / LeanIX**: Enterprise capability mapping, application dependency analysis, technology lifecycle insights
- **Miro / Lucidchart / draw.io**: Collaborative whiteboarding for design sessions, trade-off analysis, option evaluation
- **ADR Templates / Log4brains**: Architecture Decision Records with searchable portals and change history

## Architecture Analysis & Verification
- **SonarGraph / Structure101**: Dependency analysis, architectural drift detection, layered architecture enforcement
- **ArchUnit / NetArchTest**: Automated architecture rule validation integrated into CI pipelines
- **Threat Dragon / OWASP Threat Modelling**: Security threat identification, STRIDE walkthroughs, mitigation tracking
- **AWS Well-Architected / Azure Advisor / GCP Architecture Center**: Cloud architecture reviews, best-practice scoring, remediation playbooks

## Performance & Resilience Tooling
- **k6 / Gatling / JMeter**: Performance profiling tied to acceptance criteria and SLO validation
- **Chaos Mesh / Gremlin / Litmus**: Resilience experiments, failure mode exploration, steady-state hypothesis validation
- **OpenTelemetry + Grafana / New Relic**: End-to-end tracing, service health overlays, architectural bottleneck detection
- **AWS Cost Explorer / GCP Cost Intelligence**: Cost-to-serve analysis for architecture alternatives and scaling decisions

## Collaboration & Governance
- **Notion / Confluence Architecture Playbooks**: Architecture principles, guardrails, reusable patterns, integration contracts
- **Backstage / Cortex**: Service catalogues, golden path documentation, scorecards for architecture compliance
- **Jira / Azure Boards**: Architecture runway tracking, technical initiative prioritisation, risk registers
- **Slack / Teams Architecture Channels**: Design review threads, RFC management, live design clinics

## Reference Frameworks & Standards
- **Domain-Driven Design (EventStorming, Context Mapping)**: Bounded context definition, ubiquitous language sessions
- **Cloud Native Computing Foundation (CNCF) Landscape**: Technology selection for containers, service mesh, observability, and data
- **Twelve-Factor / Beyond Twelve-Factor**: SaaS application baseline, operational excellence, configuration strategies
- **ISO/IEC 42010 / TOGAF ADM**: Architecture governance frameworks for large-scale organisations
