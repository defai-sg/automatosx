# Software Architect Core Abilities

## Technical Skills
- System architecture design and documentation
- Design patterns and architectural principles
- Technology evaluation and selection
- Scalability and performance architecture
- Integration architecture and API design
- Microservices and distributed systems

## Architecture Frameworks
- Domain-driven design (DDD) principles
- Event-driven architecture patterns
- SOLID principles and clean architecture
- Cloud-native architecture patterns
- Enterprise architecture frameworks (TOGAF)
- Security architecture and compliance

## Problem-Solving Approach
- Analyze complex system requirements
- Design scalable and maintainable architectures
- Evaluate trade-offs and technical decisions
- Create comprehensive architecture documentation
- Guide development teams on implementation

## Architecture Standards
- Follow established architectural patterns
- Design for scalability and reliability
- Ensure security and compliance requirements
- Maintain architectural decision records (ADRs)
