# Universal Toolchain & Frameworks

## Planning & Alignment Layer
- **Notion / Confluence**: Single source of truth for briefs, decision records, runbooks, and onboarding playlists.
- **Jira / Linear / Asana**: Cross-team backlog visibility, sprint cadences, dependency tracking, delivery analytics.
- **Miro / FigJam / Whimsical**: Collaborative whiteboarding, system mapping, journey flows, and workshop facilitation.
- **OKR Platforms (Workboard, Perdoo)**: Goal cascading, progress reporting, and alignment rituals across functions.

## Collaboration & Communication
- **Slack / Microsoft Teams / Discord**: Channel-based communication, incident bridges, async decision threads.
- **Loom / Zoom / Google Meet**: Recorded walkthroughs, async demos, stakeholder updates, and design critiques.
- **Google Workspace / Office 365**: Shared documents, spreadsheet analytics, executive dashboards, audit trails.
- **Figma / Storybook / Lucidchart**: Visual communication, component handoff, architecture diagrams.

## Delivery & Quality Foundations
- **GitHub / GitLab / Bitbucket**: Version control, protected branches, review workflows, automation hooks.
- **CI/CD (GitHub Actions, CircleCI, Buildkite)**: Automated testing, quality gates, deployment orchestration.
- **Testing Suites (Cypress, Playwright, Jest, pytest)**: Unit/integration/E2E coverage, accessibility checks, regression safety nets.
- **Observability Stack (Datadog, Grafana, OpenTelemetry)**: Metrics, traces, logs, SLO dashboards, alert routing.

## Knowledge & Learning Systems
- **Guru / Slab / Tettra**: Short-form knowledge cards, SOP libraries, SME validation loops.
- **Dovetail / EnjoyHQ**: Research repository, insight tagging, customer signal amplification.
- **Incident.io / Rootly**: Incident command, timeline capture, post-incident learning automation.
- **Analytics Platforms (Amplitude, Mixpanel, Looker)**: Self-serve metrics, experiment readouts, KPI monitoring.

## Governance & Security
- **1Password / Vault / Boundary**: Credential management, secrets rotation, audit readiness.
- **Drata / Vanta / Tugboat**: Compliance automation, control evidence, policy tracking.
- **Workato / Zapier / n8n**: Workflow automation, integration plumbing, manual toil reduction with guardrails.

## Tooling Philosophy
- **Integrate before you proliferate**: Choose tools that share context automatically.
- **Automate the obvious**: Turn checklists into scripts and bots where repeatable.
- **Instrument everything**: Data and telemetry should be a by-product of doing the work, not an afterthought.
- **Document the why**: Pair every tool with guidance on outcomes, ownership, and escape hatches.
