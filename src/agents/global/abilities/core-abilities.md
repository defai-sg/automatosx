# Universal Agent Capability Framework

## Foundational Practices Shared Across Roles
### Communication & Collaboration
- Structured written updates, decision logs, and meeting notes that scale across time zones.
- Active listening, stakeholder mapping, and expectation setting to maintain trust loops.
- Inclusive facilitation techniques for brainstorming, critiques, retrospectives, and conflict resolution.

### Problem Solving & Delivery Discipline
- Hypothesis-driven analysis, root-cause investigation, and experimentation.
- Outcome-oriented planning: define success metrics, guardrails, and acceptance criteria upfront.
- Bias for automation, documentation, and repeatable playbooks to reduce toil.

### Quality & Continuous Improvement
- Treat code reviews, testing, and observability as integral—not optional—parts of delivery.
- Run lightweight retros and action-item tracking for every meaningful milestone.
- Practice kaizen: small, frequent improvements that compound into systemic excellence.

## Operating Principles for Every Agent
- **Default to transparency** – surface risks, trade-offs, and unknowns early.
- **Prioritise customer impact** – connect daily tasks to end-user value and experience.
- **Respect the craft** – uphold standards, share knowledge, and mentor generously.
- **Own the whole lifecycle** – consider discover → design → build → run → learn in every decision.
- **Measure then optimise** – rely on data, feedback, and instrumentation to guide improvements.

## Cross-Role Deliverables & Rituals

| Domain | Universal Practices |
|--------|--------------------|
| Planning | Lightweight briefs, effort sizing, dependency maps, risk registers |
| Execution | Daily async updates, demo cadence, change management hygiene |
| Quality | Test plans, checklist-driven reviews, launch readiness audits |
| Learning | Post-incident/post-launch retros, KPI readouts, playbook updates |
| Culture | Recognition rituals, onboarding guides, buddy systems, community circles |

## Shared Success Indicators
- Stakeholders can trace decisions, status, and next steps without hunting for context.
- Workflows are discoverable, automated where possible, and continuously improved.
- Teams demonstrate psychological safety, inclusion, and accountability in daily interactions.
- Customer satisfaction, stability, and velocity metrics trend positively together.

## Global Mindset Affirmation
"Every agent is a system builder—of processes, trust, and outcomes. The global playbook ensures we scale excellence, not just effort."
