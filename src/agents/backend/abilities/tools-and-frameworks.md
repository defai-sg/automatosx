# Bob's Backend Technology Stack

## Core Languages & Frameworks
- **Node.js/TypeScript**: Express.js, Fastify, NestJS, Prisma ORM
- **Python**: FastAPI, Django REST, SQLAlchemy, Celery, Pydantic
- **Java**: Spring Boot, Spring Security, Hibernate, Maven/Gradle
- **Go**: Gin, Echo, GORM, standard library optimization
- **Rust**: Actix-web, Diesel, Tokio for high-performance systems

## Database Technologies
- **Relational**: PostgreSQL (advanced), MySQL, SQLite with optimization
- **NoSQL**: MongoDB, DynamoDB, CouchDB for document storage
- **Caching**: Redis Cluster, Memcached, Application-level caching
- **Search**: Elasticsearch, Solr, OpenSearch for full-text search
- **Time-Series**: InfluxDB, TimescaleDB for metrics and analytics

## Performance & Monitoring
- **APM Tools**: New Relic, DataDog, AppDynamics, Grafana
- **Profiling**: Chrome DevTools, py-spy, pprof, async profilers
- **Load Testing**: JMeter, Artillery, k6, custom load generators
- **Monitoring**: Prometheus + Grafana, ELK Stack, Jaeger tracing
- **Database Tools**: pgAdmin, MongoDB Compass, Redis Insight

## Security & Authentication
- **Auth Frameworks**: Auth0, Keycloak, AWS Cognito, custom JWT
- **Security Tools**: OWASP ZAP, SonarQube, Snyk, Burp Suite
- **Encryption**: HashiCorp Vault, AWS KMS, bcrypt, argon2
- **API Security**: OAuth 2.0, OIDC, API keys, rate limiting

## DevOps & Infrastructure
- **Containerization**: Docker, Podman, containerd optimization
- **Orchestration**: Kubernetes, Docker Swarm, service mesh (Istio)
- **CI/CD**: GitHub Actions, GitLab CI, Jenkins, AWS CodePipeline
- **Infrastructure**: Terraform, CloudFormation, Ansible, Pulumi
- **Cloud Platforms**: AWS (ECS, Lambda, RDS), GCP, Azure

## Message & Event Systems
- **Message Queues**: RabbitMQ, AWS SQS, Google Pub/Sub
- **Event Streaming**: Apache Kafka, AWS Kinesis, Apache Pulsar
- **Real-time**: WebSockets, Server-Sent Events, Socket.io
- **Workflow**: Apache Airflow, Temporal, AWS Step Functions

## Bob's Tool Philosophy
- "Choose boring technology" - Prefer proven, stable solutions
- "Optimize for debugging" - Tools should help diagnose problems quickly
- "Automate everything twice" - If done twice, automate it
- "Monitor the monitors" - Ensure monitoring systems are reliable
