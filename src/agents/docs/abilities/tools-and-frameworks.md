# Documentation Specialist Tools and Frameworks

## Authoring & Content Development
- **Docs-as-code stack (Markdown, MDX, AsciiDoc)**: Version-controlled authoring aligned with engineering workflows
- **DITA XML / Oxygen**: Structured technical content, reuse, and localisation for enterprise manuals
- **Google Workspace / Microsoft 365**: Rapid drafting, reviewer-friendly change tracking, executive summaries
- **Grammarly / LanguageTool**: Automated clarity, tone, and terminology consistency checks

## API & Developer Documentation
- **Stoplight / Postman / SwaggerHub**: API modelling, contract validation, and auto-generated reference portals
- **Redoc / Slate**: Customisable developer hubs with interactive console experiences
- **Prism / Mockoon**: Mock servers to validate examples, tutorials, and request/response accuracy
- **ReadMe / Mintlify**: Hosted developer portals with metrics, changelog automation, and release notes workflows

## Knowledge Management & Governance
- **Notion / Confluence**: Single-source internal knowledge base, onboarding guides, decision records
- **GitBook / Nuclino**: Lightweight hub for customer-facing docs, tutorials, and FAQ maintenance
- **Stack Overflow for Teams / Guru**: Expert-sourced answers, knowledge verification, and content freshness alerts
- **TerminusDB / Kendra**: Semantic search and knowledge graph enrichment for large documentation estates

## Review, QA, and Accessibility
- **Vale / Alex.js**: Style linting enforcing voice, glossary, and inclusive language standards
- **Hemingway / Readable**: Readability scoring (Flesch-Kincaid, Gunning Fog) for audience-appropriate tone
- **Deque Axe / WAVE**: Accessibility audits ensuring WCAG compliance for documentation sites
- **Diff2Html / Reviewpad**: Visual diffing for efficient SME reviews and conditional approvals

## Publishing & Automation
- **Docusaurus / Astro / Next.js**: Static documentation sites with custom theme pipelines and search integrations
- **Netlify / Vercel / Cloudflare Pages**: Automated previews, environment-specific docs, and performance insights
- **GitHub Actions / CircleCI**: Continuous publishing, broken-link detection, screenshot regression suites
- **Algolia DocSearch / Typesense**: Federated search with analytics to surface content gaps and query trends
