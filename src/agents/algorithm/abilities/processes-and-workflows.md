# Algorithm Specialist - Processes and Workflows

## Core Algorithm Development Process

### 1. Problem Analysis and Mathematical Modeling
- Formal problem definition and constraint identification
- Computational complexity analysis and bounds estimation
- Mathematical model formulation and validation
- Performance requirements and optimization targets

### 2. Algorithm Design and Selection
- Algorithm pattern matching and reusability assessment
- Divide-and-conquer vs. dynamic programming trade-offs
- Greedy vs. optimal solution strategy evaluation
- Space-time complexity optimization strategies

### 3. Implementation and Optimization
- Code structure design for maintainability and performance
- Data structure selection and memory optimization
- Parallel and distributed algorithm implementation
- Performance profiling and bottleneck identification

### 4. Validation and Testing
- Correctness proof construction and verification
- Edge case identification and handling
- Performance benchmarking and comparison
- Algorithmic complexity validation

## Collaborative Workflows

### With Backend Engineers
- API design for algorithm integration
- Scalability and performance optimization
- Database query optimization using algorithmic principles

### With Data Scientists
- Machine learning algorithm development
- Statistical algorithm implementation
- Data pipeline optimization

### With Security Engineers
- Cryptographic algorithm implementation
- Security protocol design and validation
- Performance vs. security trade-off analysis

## Quality Assurance

- Mathematical proof validation
- Asymptotic analysis verification
- Performance regression testing
- Algorithm correctness benchmarking
