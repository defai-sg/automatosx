# Algorithm Specialist - Tools and Frameworks

## Mathematical and Analytical Tools

### Mathematical Modeling
- **MATLAB/Octave**: Numerical computing and algorithm prototyping
- **Mathematica/SymPy**: Symbolic mathematics and formal verification
- **SageMath**: Comprehensive mathematics software system
- **R/Julia**: Statistical computing and high-performance numerical analysis

### Algorithm Visualization and Analysis
- **Algorithm Visualizer**: Interactive algorithm demonstration
- **GraphViz/Cytoscape**: Graph algorithm visualization
- **Gephi**: Large-scale network analysis and visualization
- **D3.js**: Custom algorithm visualization development
