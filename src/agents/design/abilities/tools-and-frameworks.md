# Design Tools and Frameworks

## Product Design Stack
- **Figma / FigJam**: High-fidelity UI design, collaborative whiteboarding, design system maintenance
- **Sketch + Abstract**: Component-driven mockups with version control for macOS-centric teams
- **Adobe Creative Cloud (XD, Illustrator, Photoshop)**: Brand asset creation, motion studies, marketing collateral support
- **Framer / ProtoPie**: Interactive prototyping for advanced micro-interactions and usability validation

## Research & Validation Platforms
- **Maze / UserTesting**: Remote usability testing, task analysis, qualitative insight capture
- **Dovetail / Aurelius**: Research repository and tagging for synthesis, insight management, and trend tracking
- **Hotjar / FullStory**: Behavioral analytics, heatmaps, and session replay to validate design hypotheses
- **Optimal Workshop**: IA validation through card sorting, tree testing, and navigation benchmarking

## Accessibility & Quality Tooling
- **Stark / Axe / Lighthouse**: Automated accessibility audits against WCAG 2.2 AA requirements
- **Color Oracle / Contrast**: Visual impairment simulation and contrast checking in the design phase
- **Storybook Accessibility Add-ons**: Component-level accessibility regression coverage in the codebase
- **NVDA / VoiceOver**: Screen reader validation for critical flows and assistive technology parity

## Collaboration & Dev Handoff
- **Notion / Confluence**: Design decision logs, research reports, and alignment with product briefs
- **Zeplin / Storybook**: Token-aligned developer handoff, redlines, and component usage guidance
- **Jira / Linear**: Design backlog, sprint alignment with engineering, acceptance criteria tracking
- **Slack / Loom**: Asynchronous feedback loops, design critiques, and stakeholder alignment

## Design System Operations
- **Design Tokens (Style Dictionary, Theo)**: Source-of-truth configuration for cross-platform theming
- **Figma Libraries + Tokens Studio**: Multi-brand design system governance and documentation
- **Chromatic**: Visual regression testing synchronized with component library releases
- **Backlight / ZeroHeight**: Living documentation that links design intent with implementation details
