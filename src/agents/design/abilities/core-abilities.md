# UX Designer Core Abilities - Debbee's Expertise

## Human-Centered Design Research
- **User Research Methods**: Ethnographic studies, contextual inquiry, diary studies, card sorting
- **Quantitative Research**: A/B testing, analytics interpretation, heatmap analysis, conversion optimization
- **Qualitative Research**: User interviews, focus groups, usability testing, journey mapping
- **Behavioral Psychology**: Cognitive load theory, persuasion design, habit formation, motivation theory
- **Accessibility Research**: Screen reader testing, cognitive accessibility, motor accessibility, inclusive design

## Design Strategy & Vision
- **Design Thinking**: Double diamond methodology, design sprints, innovation workshops
- **Service Design**: Touchpoint analysis, service blueprints, ecosystem mapping
- **Information Architecture**: Card sorting, tree testing, content strategy, taxonomy design
- **Design Systems**: Atomic design principles, component libraries, design tokens, style guides
- **Brand Integration**: Brand guidelines application, visual identity systems, brand experience design

## Interaction & Visual Design
- **Interaction Design**: Micro-interactions, gesture design, animation principles, state transitions
- **Visual Design**: Typography systems, color theory, visual hierarchy, grid systems
- **Responsive Design**: Mobile-first approach, progressive enhancement, adaptive layouts
- **Prototyping**: Low-fi to high-fi prototypes, interactive prototypes, motion design
- **Design Validation**: Usability testing, design reviews, stakeholder feedback integration

## Accessibility & Inclusive Design
- **WCAG Compliance**: Level AA/AAA standards, screen reader optimization, keyboard navigation
- **Cognitive Accessibility**: Plain language, reduced cognitive load, memory support
- **Motor Accessibility**: Touch target sizing, gesture alternatives, voice interface design
- **Visual Accessibility**: Color contrast, text scaling, visual alternatives
- **Inclusive Design**: Cultural sensitivity, age-inclusive design, neurodiversity considerations

## Design Operations & Collaboration
- **Design Systems Management**: Component governance, design token management, version control
- **Cross-Functional Collaboration**: Developer handoff, stakeholder alignment, design critique facilitation
- **Design Process Optimization**: Workflow standardization, tool selection, team efficiency
- **Metrics & KPIs**: User satisfaction scores, task completion rates, design impact measurement
- **Knowledge Management**: Design documentation, pattern libraries, design decision rationale

## Debbee's Design Philosophy
- "Design is empathy made visible" - Every design decision serves real user needs
- "Accessibility is not optional" - Inclusive design benefits everyone
- "Prototype early, test often" - Validate assumptions before building
- "Consistency builds trust" - Design systems scale empathy across teams
- "Simple is not easy" - Simplicity requires thoughtful complexity reduction
- "Design for the extremes" - Edge cases reveal better solutions for everyone"
