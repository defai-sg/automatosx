# Business Analyst Tools and Frameworks

## Data Analysis & Statistical Tools
- **Statistical Software**: R, Python (pandas, NumPy, SciPy), SPSS, SAS, Stata
- **Data Visualization**: Tableau, Power BI, Qlik Sense, D3.js, matplotlib, seaborn, ggplot2
- **Spreadsheet Analysis**: Advanced Excel, Google Sheets, macro development, pivot tables
- **Database Tools**: SQL (PostgreSQL, MySQL, SQLite), NoSQL (MongoDB), query optimization
- **Big Data Tools**: Apache Spark, Hadoop, Hive, data lakes, distributed computing

## Market Research & Survey Tools
- **Survey Platforms**: Qualtrics, SurveyMonkey, Google Forms, Typeform, LimeSurvey
- **Market Research**: Nielsen, Euromonitor, IBISWorld, Statista, Mintel, Forrester Research
- **Consumer Insights**: Google Analytics, Facebook Insights, social media analytics, sentiment analysis
- **Competitive Intelligence**: SEMrush, Ahrefs, SimilarWeb, Crunchbase, PitchBook
- **Interview Tools**: Zoom, Microsoft Teams, transcription software, qualitative analysis tools

## Business Intelligence & Analytics Platforms
- **BI Platforms**: Microsoft Power BI, Tableau, QlikView, Looker, Sisense, IBM Cognos
- **Enterprise Analytics**: SAP BusinessObjects, Oracle Analytics Cloud, SAS Visual Analytics
- **Cloud Analytics**: Amazon QuickSight, Google Data Studio, Azure Analytics, AWS Analytics
- **Reporting Tools**: Crystal Reports, SSRS, automated reporting, dashboard development
- **Data Warehousing**: Snowflake, Amazon Redshift, Google BigQuery, data modeling tools

## Financial Analysis & Modeling Tools
- **Financial Modeling**: Excel financial models, Monte Carlo simulation, sensitivity analysis
- **Investment Analysis**: Bloomberg Terminal, Reuters Eikon, FactSet, capital budgeting tools
- **Accounting Software**: QuickBooks, SAP, Oracle Financials, financial statement analysis
- **Economic Analysis**: FRED (Federal Reserve Economic Data), World Bank data, economic modeling
- **Valuation Tools**: DCF models, comparable company analysis, precedent transaction analysis

## Strategic Planning & Framework Tools
- **Strategy Frameworks**: SWOT analysis, Porter's Five Forces, BCG Matrix, McKinsey 7S Framework
- **Business Model Tools**: Business Model Canvas, Lean Canvas, value proposition design
- **Strategic Planning**: Balanced Scorecard, OKRs, strategic roadmapping, scenario planning
- **Risk Management**: Risk registers, Monte Carlo risk analysis, decision trees, sensitivity analysis
- **Competitive Analysis**: Competitor analysis templates, market positioning maps, competitive matrices

## Presentation & Communication Tools
- **Presentation Software**: PowerPoint, Google Slides, Prezi, Keynote, advanced presentation design
- **Design Tools**: Canva, Adobe Creative Suite, Figma, presentation templates, infographic creation
- **Report Writing**: LaTeX, advanced Word processing, executive summary templates, report automation
- **Data Storytelling**: Narrative techniques, insight communication, persuasive presentation methods
- **Client Communication**: CRM systems, client portals, stakeholder communication plans
